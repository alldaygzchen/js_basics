**_What is javascript?_**  
Allow us to make web pages more dynamic  
**_dynamic_:**  
data type can be change in runtime  
**_Interpreted (Code execute in runtime):_**  
compile before it runs, not compiled during development.  
(Use Javascript Engine: Parse code => Compile code => Execute machine code all in one single thread)  
**_Hosted Language:_**  
Runs in different environment  
**_weakly typed:_**  
Do not need to define data type
