**_Debugging:_**

- Chrome dev tool

  - Breakpoint stop the execution in the point which is to reach
  - Resume script execution: run to next breakpoint
  - Step over new function call: run to next line
  - Step into next function call: jump to function definition
  - Step out function call: jump out function definition
  - Step : Step over new function call + Step into next function call
  - Scope and watch can keep track in code executiom
  - Exist conditional breakpoint and addEventListner breakpoint
